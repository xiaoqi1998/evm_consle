测试恢复功能

PC端：上传图片识别
手机端：摄像头扫码

技术说明

1. 二维码生成

使用 qrcode.js 库在前端生成二维码，数据格式：

{ "v": 1, "a": "账户别名", "s": "切片 C 内容" }
2. PC端识别

使用 jsQR 库解析上传的二维码图片：

  • 通过 FileReader 读取图片文件
  • 将图片绘制到 canvas
  • 使用 jsQR 解析二维码数据
3. 手机端扫码

使用 html5-qrcode 库调用摄像头：

  • 请求摄像头权限
  • 实时扫描视频流
  • 识别二维码并返回数据
重要：所有二维码生成和解析逻辑都在前端完成，绝不会将切片 C 发送到服务器。
