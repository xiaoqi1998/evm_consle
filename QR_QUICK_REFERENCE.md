# 🚀 二维码密保体系 - 快速参考指南

## 📱 快速开始

### 备份账户
1. 登录账号
2. 点击"添加账户"
3. 输入私钥和地址
4. 点击"保存账户"
5. ✅ **系统自动生成密保二维码**
6. 点击"下载密保图片"
7. 打印或存入物理加密U盘

### 恢复账户
1. 登录账号
2. 找到需要恢复的账户
3. 点击"恢复"按钮
4. 选择恢复方式：
   - **PC端**：上传二维码图片
   - **手机端**：开启摄像头扫码
5. ✅ **系统自动同步并重新分片**
6. 设置新密码

## 🔧 技术命令

### 生成二维码
```javascript
const cardHTML = await QRGenerator.generateBackupCard(alias, sliceC, address);
QRGenerator.downloadBackupImage(alias);
```

### 扫码恢复
```javascript
// PC端
const qrData = await QRScanner.scanImageFromFile(file);

// 手机端
const qrData = await QRScanner.scanWithCamera();

// 解析
const parsed = QRScanner.parseQRData(qrData);
```

## 📊 数据格式

### 二维码内容
```json
{
  "v": 1,
  "a": "账户别名",
  "s": "切片 C 内容"
}
```

### 分片结构
- **切片 A**：服务器存储（明文）
- **切片 B**：本地存储（AES 加密）
- **切片 C**：二维码存储（密码分片）

### 还原公式
```
私钥 = Shamir.combine([切片 A, 切片 C])
```

## ⚠️ 安全检查清单

### 备份时
- [ ] 二维码已下载
- [ ] 已打印或存入加密U盘
- [ ] 未存入手机相册
- [ ] 未存入云端
- [ ] 已核对账户别名和地址

### 恢复时
- [ ] 二维码未泄露
- [ ] 二维码未损坏
- [ ] 账户别名匹配
- [ ] 已设置新密码
- [ ] 已验证账户可用

## 🛠️ 故障排除

### 问题：二维码生成失败
**原因**：qrcode.js 未加载
**解决**：刷新页面，检查网络连接

### 问题：图片识别失败
**原因**：图片不清晰或二维码损坏
**解决**：重新生成二维码，使用高分辨率图片

### 问题：摄像头无法使用
**原因**：浏览器权限未授予
**解决**：点击地址栏的锁图标，允许摄像头访问

### 问题：切片不匹配
**原因**：二维码中的别名与账户别名不一致
**解决**：确保使用正确的二维码
