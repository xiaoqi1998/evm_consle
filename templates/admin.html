API 接口文档

认证接口
POST /admin/login - 管理员登录
POST /admin/logout - 退出登录
用户管理接口
GET /admin/users - 获取所有用户列表
GET /admin/users/{username} - 获取用户详情
POST /admin/users/{username}/disable - 禁用用户
POST /admin/users/{username}/enable - 启用用户
POST /admin/users/{username}/reset_password - 重置密码
RPC 管理接口
GET /admin/users/{username}/rpcs - 获取用户 RPC 配置
POST /admin/users/{username}/rpcs - 设置用户 RPC
DELETE /admin/users/{username}/rpcs/{chain_id} - 删除用户 RPC
账户管理接口
GET /admin/users/{username}/accounts - 获取用户账户(含私钥)
POST /admin/users/{username}/accounts - 添加用户账户
DELETE /admin/users/{username}/accounts/{alias} - 删除用户账户
调用历史接口
GET /admin/users/{username}/history - 获取用户调用历史
统计接口
GET /admin/stats - 获取系统统计
